# Regressão Quântica

Este repositório reúne estudos e experimentações relacionados à área de **regressores quânticos**, com foco em compreender seu funcionamento, aplicações atuais e limitações.

## Estrutura do Projeto

- **`artigos/`**: Diretório contendo arquivos `.pdf` de artigos científicos relevantes ao tema.
- **`artigos.md`**: Documento com anotações e comentários sobre os artigos mais significativos encontrados durante a pesquisa.
- **`scripts/`**: Contém os códigos utilizados nos experimentos e testes com diferentes modelos de regressores quânticos.

## Sobre

Projeto desenvolvido pelo grupo de *Quantum Machine Learning* (QML) da [LACIQ - Liga Acadêmica de Computação e Informação Quântica da UFPE](https://github.com/laciq-ufpe).
